# Project Theme

This file defines the current learner persona and content theme for lesson-authoring agents.

## Current Theme

- **Audience:** 3rd-grade learner.
- **Persona:** likes princesses, dress-up, fairy tales, castles, jewels, animals, magic, and playful fantasy worlds.
- **Tone:** warm, encouraging, concrete, and short.
- **Examples:** crowns, gowns, shoes, royal jewels, treasure bags, castle doors, magic spinners, royal carnival games, dragons, knights, fairies.
- **Instruction style:** concrete interaction first, math vocabulary second.

## Theme Rules

- Keep language age-appropriate and visual.
- Prefer story contexts the learner can imagine manipulating.
- Use math terms clearly, but only after the concrete example motivates them.
- Avoid making the theme feel babyish; frame activities as design, puzzles, challenges, and royal academy missions.

## Future Theme Changes

To change the curriculum theme later, update this file first.

Skills and agents should read this file instead of hardcoding princess/royal/fantasy assumptions.
